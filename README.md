# Contextual-Chatbot-Training
Training a keras model to try to figure out what user intends (in which sense or proportion the user is asking a question).

## Requirements
- Python 3 or higher.

## Packages

### Tensorflow
```bash
py -m pip install tensorflow
```
### Numpy
```bash
py -m pip install numpy
```

## Start Training
```bash
py training.py
```
